# Netflix Clone

This project is a recreation of the Netflix landing page built as part of my frontend development journey.

The goal of this project was to practice HTML, CSS, Flexbox, positioning, and animations while building a real-world user interface.

## Features

* Netflix-inspired homepage design
* Navigation bar with buttons
* Hero section with email input
* CSS animations and transitions
* Clean and organized file structure

## Built With

* HTML5
* CSS3
* Flexbox
* CSS Animations

## What I Learned

While building this project, I practiced:

* Structuring web pages with HTML
* Styling layouts with CSS
* Using Flexbox for alignment
* Working with transforms and animations
* Organizing project files
* Using Git and GitHub
